# Ames Housing Dataset

## Описание проекта

Данный проект использует **Ames Housing Dataset** набор данных о продаже жилой недвижимости в городе Эймс (штат Айова, США).

Набор данных содержит информацию о **1460 проданных домах** и включает **79 признаков**, описывающих различные характеристики объектов недвижимости.

Датасет широко применяется в задачах **машинного обучения**, **регрессионного анализа**, **анализа данных** и **построения моделей прогнозирования стоимости жилья**.


## Информация о датасете

### Основные характеристики

| Параметр | Значение |
|----------|----------|
| Количество объектов | 1460 |
| Количество признаков | 79 |
| Целевая переменная | `SalePrice` |
| Тип задачи | Регрессия |
| Типы данных | Числовые и категориальные |

### Целевая переменная

`SalePrice` - фактическая цена продажи дома.

### Примеры признаков

Датасет содержит информацию о:

- площади земельного участка;
- местоположении дома;
- годе постройки;
- качестве строительства;
- количестве комнат;
- количестве ванных комнат;
- площади подвала;
- наличии гаража;
- состоянии объекта недвижимости;
- дополнительных характеристиках здания.

---

## Цель проекта

Основная задача проекта это построение модели машинного обучения для прогнозирования стоимости недвижимости на основе характеристик домов.

В рамках проекта могут выполняться следующие этапы:

1. Предварительный анализ данных (EDA).
2. Очистка и обработка пропущенных значений.
3. Кодирование категориальных признаков.
4. Масштабирование данных.
5. Отбор наиболее информативных признаков.
6. Модель с наилучшими показателями
---

## Области применения

Датасет часто используется для:

- обучения методам анализа данных;
- изучения регрессионных моделей;
- инженерии признаков (Feature Engineering);
- сравнения алгоритмов машинного обучения;
- участия в соревнованиях по анализу данных.
