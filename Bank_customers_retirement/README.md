# Customer Retirement Prediction

## Описание проекта

Цель проекта-построить модель машинного обучения для прогнозирования выхода клиента на пенсию на основе его демографических и финансовых характеристик.

## О датасете

Датасет содержит информацию о клиентах и их пенсионных накоплениях. Каждая строка представляет отдельного клиента, а признаки описывают его возраст и объем средств, накопленных на пенсионном счете.

Набор данных используется для решения задачи прогнозирования факта выхода на пенсию. На основе имеющихся характеристик модель должна определить, относится ли клиент к категории пенсионеров.

### Признаки

| Переменная   | Описание                                                                       |
| ------------ | ------------------------------------------------------------------------------ |
| Customer ID  | Уникальный идентификатор клиента                                               |
| Age          | Возраст клиента                                                                |
| 401k_savings | Сумма накоплений на пенсионном счете 401(k)                                    |
| Retire       | Целевая переменная: 0-клиент продолжает работать, 1-клиент вышел на пенсию |

## Результат

В результате проекта была построена модель классификации, способная прогнозировать вероятность выхода клиента на пенсию на основе его возраста и размера пенсионных накоплений.
Все модели отлично отрабатывают и имеют хорошие показатели , отсутствие переобучения и быструю предективную скорость.
Переменная "SavingsPerAge" имеет положительное влияние на некоторые модели , улучшая некоторые метрики. Из-за малого количества переменных и данных не удалось преодолеть accuracy выше 95%. 
Вероятно лучшей моделью и более сбалансированной является **RandomForest**
