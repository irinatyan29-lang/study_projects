# Enron Email Analysis & Classification 

Проект по обработке естественного языка (NLP), анализу социальных графов и машинному обучению на основе знаменитого набора данных **Enron Email Dataset**.

## 📌 Описание проекта

Проект посвящен извлечению инсайтов, анализу скрытых связей и классификации текстовых сообщений из архива переписки топ-менеджеров компании Enron. Набор данных уникален тем, что содержит реальную, а не синтетическую бизнес-коммуникацию, отражающую корпоративную культуру и паттерны поведения сотрудников в кризисный период.

В зависимости от архитектуры, данный проект решает задачи:
*   **NLP / Text Classification:** Поиск спама, классификация писем по категориям или анализ тональности (Sentiment Analysis).
*   **Anomaly Detection:** Выявление аномального поведения сотрудников или подозрительных цепочек писем.
*   **Network Analysis:** Визуализация графа коммуникаций внутри организации.

## 📊 Описание датасета

Оригинальный массив данных содержит около **500 000 писем** от более чем **150 сотрудников** (в основном высшего руководства Enron).

Каждое письмо представлено в виде текстового файла и содержит:
*   `Date`: Дата и время отправления.
*   `From` / `To` / `Cc` / `Bcc`: Электронные адреса отправителя и получателей.
*   `Subject`: Тема письма.
*   `X-Folder` / `X-Origin`: Структура папок в почтовом клиенте (например, *discussion_threads, invoices, internal*).
*   `Body`: Полное текстовое содержимое письма.

> **Особенность данных:** Набор данных требует глубокой предобработки (очистка от дубликатов, автоматических подписей, системных логов и HTML-тегов).

## 🛠️ Стек технологий

*   **Язык:** Python 3.x
*   **Предобработка и NLP:** NLTK, Spacy, Regular Expressions (`re`)
*   **Анализ текста:** Scikit-learn (`TfidfVectorizer`, `CountVectorizer`)
*   **Машинное обучение:** Naive Bayes, Logistic Regression, Support Vector Machines (LinearSVC)
*   **Глубокое обучение (опционально):** Transformers (Hugging Face / BERT) для тематического моделирования
*   **Анализ графов:** NetworkX (для построения связей между сотрудниками)

## 🚀 Как запустить проект

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   cd название-репозитория
   ```

2. Скачайте оригинальный датасет (из-за большого размера файл `emails.csv` обычно исключается из git через `.gitignore`):
   * Загрузите CSV-версию с Kaggle или архив с сайта CMU.
   * Поместите его в папку `data/`.

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Запустите ноутбук:
   ```bash
   jupyter notebook enron_analysis.ipynb
   ```
